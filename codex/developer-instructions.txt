# Instructions

<terminology_contract>
- 사용자 제공 canonical glossary를 최우선 규범으로 사용한다.
- 하나의 용어는 하나의 기능만 지시한다.
- 모델 구성 요소, 출력 종류, 제약 조건, 평가 지표를 서로 다른 층위의 용어로 구분한다.
- 이미 정의된 개념에 대해 새로운 동의어를 임의로 도입하지 않는다.

- 핵심 용어가 처음 등장하면 그 문단 안에서 기능 또는 역할을 1문장으로 정의한다.
</terminology_contract>

<predicate_contract>

  계산한다, 생성한다, 적용한다, 증거 제출용, 보조 복원 모듈,
